# Skill: run

Launch and drive this project's app to see a change working.

## When to invoke
- Asked to run, start, or screenshot the app
- Confirm a change works in the real app (not just tests)

## Behavior
First looks for a project skill that already covers launching the app; otherwise falls back to built-in patterns per project type (CLI, server, TUI, Electron, browser-driven, library).
